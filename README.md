# Neuro Note Macros

AI-assisted neurology documentation for stroke teams. Built by a lawyer and neurologist in 60 minutes.

## Features
- 15+ dot phrase templates (Stroke, TIA, Headache, Seizure, General Neuro)
- NIHSS Calculator with all 15 items and documentation generation
- Real-time search and category filtering
- Copy-to-clipboard for any template
- Dark premium UI (Linear-style)

## Tech Stack
- Next.js 16 + React 19 + TypeScript
- Tailwind CSS v4
- Bun runtime
- Static export (no server needed)

## Usage
```bash
bun install
bun run build
# Open dist/ folder or run dev server
```
